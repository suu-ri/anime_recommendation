# 課題8-anime recommendation

アニメおすすめアプリ

## 課題内容（どんな作品か）

- アニメおすすめ投稿・閲覧アプリ
- ユーザーがアニメのタイトル、ジャンル、あらすじ、評価を投稿できる
- 投稿されたアニメは一覧で表示され、評価順にソートできる
- 投稿したアニメは編集・削除が可能
- ユーザー登録、ログイン機能


## DEMO

https://lichongyu.sakura.ne.jp/anime_recommendation
